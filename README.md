
TrendScale Pro Oscillator combines three powerful momentum tools — RSI, SDO (Stochastic-Donchian Oscillator), and EOT (Ehlers-based Oscillator Technique) — into a single, unified indicator.
Switch between individual modes to analyze each oscillator in depth, or use Combined Mode to view all three normalized to the same scale with a dynamic flow cloud that visualizes momentum agreement and divergence at a glance.
Key Features:

RSI with divergence detection and pivot labels
SDO blending Stochastic and Donchian channel momentum, with signal line, zone boxes, and divergence detection
EOT using a high-pass filter + Super Smoother for clean cycle analysis, with tiered entry signals, exhaustion detection, bounce alerts, and pressure circles
Combined Mode with confluence scoring — a ★ LONG or ★ SHORT label fires when 2 or more oscillators agree
Optional Higher Timeframe (HTF) mode for all calculations
Full alert support across all modes

Ideal for traders who want multi-method confirmation without cluttering their chart with multiple separate indicators.
